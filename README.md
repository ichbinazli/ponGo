1. **Projeyi klonlayın**
   ```bash
   git clone https://github.com/ichbinazli/ft_transcendence.git
   cd ft_transcendence/ft_transcendence-frontend/ft_transcendence-frontend
   ```

2. **Bağımlılıkları yükleyin**
   ```bash
   npm install
   ```

3. **Geliştirme sunucusunu başlatın**
   ```bash
   npm run dev
    ya da
   npm run build
   npm run serve
   ```

4. **Tarayıcıda açın**
   ```
   http://localhost:5173
   ```
