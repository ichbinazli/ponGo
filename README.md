# 🌐 ponGo Project

> Modern web teknolojileri kullanılarak geliştirilen, gerçek zamanlı etkileşim sunan bir full-stack uygulama.

---

## 🚀 Proje Hakkında
Kullanıcıların gerçek zamanlı olarak etkileşim kurabildiği bir platformdur. Proje kapsamında;

* 🔐 Kullanıcı kayıt & giriş sistemi
* 🎮 Gerçek zamanlı multiplayer Pong oyunu
* 💬 Anlık mesajlaşma sistemi
* 👤 Kullanıcı profil yönetimi
* 🤝 Matchmaking sistemi

gibi özellikler geliştirilmiştir.

---

## 🧱 Kullanılan Teknolojiler

### Backend

* Node.js
* Express.js
* RESTful API mimarisi
* WebSocket (gerçek zamanlı iletişim)

### Frontend

* HTML5
* CSS3
* JavaScript / TypeScript
* Modern UI/UX tasarımı

### Veritabanı

* PostgreSQL / SQL

### DevOps & Araçlar

* Docker
* Nginx
* Git & GitHub

---

## ⚙️ Özellikler

✔️ Güvenli kullanıcı kimlik doğrulama (JWT)
✔️ Gerçek zamanlı oyun ve sohbet deneyimi
✔️ Responsive (mobil uyumlu) arayüz
✔️ Kullanıcı profilleri ve istatistikler
✔️ Ölçeklenebilir mimari
